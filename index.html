8. Fraud Investigation Copilot — LangChain + FastAPI (RAG + Similar Case Retrieval)

Policy Ingestion:

  • Ingested .txt policy files from data/policies/
  • Chunked using RecursiveCharacterTextSplitter
  • Embedded using OpenAIEmbeddings and stored in Chroma (vectorstore/policies)

  • Loaded creditcard.csv from data/credit/ and filtered Class == 1 (confirmed fraud)
  • Converted each row to a concise text summary
  • Embedded and stored in a separate Chroma store (vectorstore/fraud_cases)

Inference Flow (POST /explain):

  • Receive transaction JSON
  • Use policy RAG to determine applicable rules and generate explanation
  • Retrieve similar historical fraud cases
  • Extract rule IDs (e.g., AML-TRX-001) using regex
  • Return structured JSON: decision + explanation + rules + similar cases

Tech Stack: Python, FastAPI, LangChain, Chroma, Pandas, python-dotenv, OpenAPI/Swagger.

6. Kidney CT Image Classification — PyTorch + EfficientNet (Transfer Learning)

Built a clinically meaningful image classification system to classify kidney CT scan images into four classes: Normal, Cyst, Stone, and Tumor. The approach uses transfer learning and class-imbalance aware training with robust evaluation.

Approach & Methodology:

  • Preprocessing: Resized to 224×224; normalization using ImageNet mean/std
  • Augmentation: Rotation + affine transforms applied only to training data (no augmentation on validation/test)
  • Split: Train 70% / Validation 15% / Test 15%
  • Model: EfficientNet backbone pretrained on ImageNet + custom 4-class head
  • Two-Phase Training: (1) Train classifier head with frozen backbone (2) Fine-tune last backbone layers with lower LR
  • Class Imbalance: Class-weighted CrossEntropyLoss to improve minority-class recall

Evaluation Focus: per-class recall (especially Tumor), macro F1-score, precision, confusion matrix (accuracy used as secondary).

Results Summary: Overall Accuracy ~97%, Tumor Recall ~96%, Macro F1-score ~96%.

Python PyTorch Torchvision EfficientNet Transfer Learning Class Weights Confusion Matrix
